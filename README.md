# Mailedom Content Delivery Network

## Access Images

You can use `jsdelivr` to access the images via CDN.

It's highly recommended to do so.


## Vehicle Example

```
https://cdn.jsdelivr.net/gh/kuerton/cdn.mailedom-files/img/main/vehicles/(name)/(name).webp
```
